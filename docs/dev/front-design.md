## 🎨 设计规范

### 配色方案

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主背景 | `#f5f7fa` | 浅灰色背景 |
| 卡片背景 | `#ffffff` | 纯白色 |
| 主色调 | `#3b82f6` → `#2563eb` | 蓝色渐变 |
| 警告色 | `#f59e0b` → `#d97706` | 橙色渐变（绑定设备） |
| 文字主色 | `#1f2937` | 深灰色 |
| 文字次要 | `#6b7280` | 中灰色 |
| 边框 | `#e5e7eb` | 浅灰色 |

### 圆角规范

- 大卡片：`12px`
- 按钮：`10px`
- 小元素：`6-8px`
- 标签：`4px`

### 间距规范

- 页面边距：`20px`（移动端 `12px`）
- 卡片内边距：`32px`（移动端 `20px`）
- 元素间距：`16-24px`

### 阴影规范

- 卡片阴影：`0 2px 8px rgba(0, 0, 0, 0.08)`
- 按钮阴影：`0 4px 12px rgba(59, 130, 246, 0.3)`
- 悬停阴影：`0 6px 16px rgba(59, 130, 246, 0.4)`

## 🚀 使用方法

### 开发环境

1. **启动后端**
   ```bash
   cd /Users/xx/ck/code/fenfa
   go run ./cmd/server
   ```

2. **启动前台开发服务器**
   ```bash
   cd web/front
   npm run dev
   ```

3. **访问**
   ```
   http://127.0.0.1:5173/apps/{APP_ID}
   ```

### 生产环境

1. **构建前台**
   ```bash
   cd web/front
   npm run build
   ```

2. **启动后端**（会自动加载构建后的静态文件）
   ```bash
   go run ./cmd/server
   ```

3. **访问**
   ```
   http://127.0.0.1:8000/apps/{APP_ID}
   ```
