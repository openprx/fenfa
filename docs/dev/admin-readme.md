# 分发管理后台

现代化的移动应用分发管理后台界面。

## 功能特性

### 🔐 登录认证
- Token 认证机制
- 自动保存登录状态
- 安全退出登录

### 📱 应用管理
- 查看所有已上传的应用
- 搜索应用（支持名称、Bundle ID、Application ID）
- 快速跳转到统计和事件页面

### ⬆️ 上传构建
- 支持 iOS (IPA) 和 Android (APK) 文件上传
- 完整的元数据配置（版本号、构建号、渠道等）
- 实时上传进度反馈
- 上传成功后显示下载链接

### 📊 下载统计
- 按应用查看各版本下载数据
- 清晰的数据展示表格

### 📋 访问明细
- 查看用户访问和下载事件
- 支持按应用、事件类型、时间范围筛选
- 详细的用户代理信息解析

### 💾 数据导出
- 导出 Releases CSV
- 导出 Events CSV
- 导出 iOS 设备列表 CSV
- 支持时间范围和应用筛选

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **TypeScript** - 类型安全
- **现代 CSS** - 渐变、阴影、动画效果

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
